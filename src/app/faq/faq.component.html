What is QtRpi?

QtRpi is a complete bundle to create Qt application for the Raspberry Pi. This bundle is composed by several parts:

  • Preconfigured Qt version for your computer
  • Precompiled Qt binaries for your Raspberry Pi
  • The toolchain
  • A minimal raspbian sysroot

How to install QtRpi?

QtRpi requires several tasks to be installed properly on both your computer and your Raspberry Pi. To facilitate the installation, we created a script to automatically perform all steps for you. This script init-qtrpi-minimal.sh can be found on our QtRpi repository on GitHub.

Clone the repository on your computer 

git clone https://github.com/neuronalmotion/qtrpi.git

or download the ZIP file.

Configure your environement variables

The script support several versions of Qt and Raspberry Pi. If you want install a specific version of QtRpi, you should set the following environement variales:

Name Description Values Default value
QTRPI_QT_VERSION The version of the Qt framework 5.7 5.7
QTRPI_TARGET_DEVICE The version of the Raspberry Pi board linux-rasp-pi2-g++ linux-rasp-pi2-g++
QTRPI_TARGET_HOST The Raspberry Pi SSH host name. Generally the IP address of your board. For example: pi@192.168.1.5

Run the script

Check your environement variables describes above before running the script. You should have 

cd qtrpi
./init-qtrpi-minimal.sh

Check your installation

On your computer, you have a folder /opt/qtrpi with:

  • bin : qmake-rpi a symbolic link to the prefconfigured qmake
  • raspbian : the sysroot
  • raspi : the toolchain, the preconfigured Qt, the precompiled Qt for Raspberry Pi

On your Raspberry Pi, you should get a folder /local/usr/qt5pi with the the precompiled Qt for Raspberry Pi.

Export the QtRpi bin path

Add the folder /opt/qtrpi/bin to yout PATH. 

export PATH=/opt/qtrpi/bin:$PATH # Add this line to your .bashrc
source ~/.bashrc

How to use QtRpi?

Now that QtRpi is installed on your computer and deployed to your Raspberry Pi, you can cross-compile a Qt application. You can now easily cross-compile your project with qmake-rpi. 

cd myproject
qmake-rpi
make
make install

The generated binary file is ready to be copy and executed on your Raspberry Pi!
