Purpose

To deploy Qt applications on a Raspberry Pi, you need to have cross-compiled your Qt dependencies. Today, this has to be done by hand (see here, here and even here). We offer you on this website: Raspberry Pi precompiled Qt binaries with all the dependencies.

Supported devices

We provide cross-compiled binaries for all existing Raspberry Pi versions:

  • Raspberry Pi 1
  • Raspberry Pi 2
  • Raspberry Pi 3

Download

Go to the Download page and start getting you needed version of Qt. Right now we support:

  • Qt 5.7.0
  • Qt 5.6.2 (LTS)

About us

Our company, NeuronalMotion, works heavily with Qt and in some of our projects, we needed to deploy Qt applications to a Raspberry Pi. Because we are developers and like to be lazy, we decided to store the precompiled Qt dependencies on a website, rather than recompiling them each time.
